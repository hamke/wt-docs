워드프레스 1:1 개인 레슨 상세 정보

업데이트 : --01

워드프레스(WordPress)는 흔히 알려져 있는 것과는 달리, 웹빌더 서비스가 아닌 원래 개발(Web Development)의 영역이며 학습해야 하는 내용이 상당히 방대하고 깊히 알수록 어려워 지기 때문에, 초보자의 경우 처음에 제대로 핵심 구동 원리를 배우는 것이 매우 중요합니다.

워드프레스톡(WP-TALK)이 제공하는 1:1 레슨의 경우, 이와 같은 내용을 학습자의 사전 지식과 원하는 학습 목적에 맞춰 최적화된 내용을 빠르게 교육 받을 수 있는 장점이 있습니다.

◼︎ 커리큘럼




- 워드프레스 프레임워크의 특징 및 핵심 구동 원리

- 워드프레스 파일 구조 및 용도

- 워드프레스 테마 및 플러그인 구동 원리 및 사용 방법

- 워드프레스 템플렛 구조의 이해 ( Template Hierarchy )

- 워드프레스 커스터마이징 방법 ( functions.php / Hooks )

- 워드프레스 테마 디자인/수정 방법 ( 자바스크립트 / CSS 및 인트로 페이지 수정 )

- 워드프레스 포스트 타입 활용 방법 ( Custom Post Type )

- 데이터베이스의 활용 ( $wpdb / WP_Query / Template Tags )

위의 내용은 워드프레스(WordPress)를 다룰 때 반드시 알아야 할 기본적인 개념들이며, 교육 과정에서 사이트 운영 목적 및 학습자의 상황에 따라 유동적으로 조정될 수 있습니다.

◼︎ 학습 시간 및 비용

1:1 레슨의 경우, 전체 학습 시간은 자유롭게 선택하실 수 있습니다.

기본 3시간이며, 필요시 2시간 단위로 추가될 수 있습니다. (우선 기본 3시간을 추천 드립니다.)

(프로그래밍은 실전 예제 학습이 매우 중요하며 ,동일한 레슨 범위더라도 예제에 따라 학습 시간을 유동적으로 조정할 수 있습니다.)

비용은 기본 3시간에 30만원, 추가 2시간에 15만원입니다. (투자된 금액 대비 학습 후 이를 활용한 효과는 분명 10배(300만원) 이상의 가치가 있을 것입니다. 2인 이상 단체의 경우, 할인될 수 있습니다.)

◼︎ 교육 진행 방식

기본적으로 저희가 제공해 드리는 온라인 사이트 (설치 없음, 웹브라우저에서 접속)에서 화상 교육 방식으로 진행합니다.

만일 위치가 서울 또는 근교이시고 교육에 적합한 오프라인 장소가 준비되어 있으시다면, 오프라인 교육도 가능합니다.
